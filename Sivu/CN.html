About the Game

The game project is a 2D shooter. The player has to defeat oncoming enemies to prevent them from corrupting the players code.

It is highly inspired by Space Invaders and made using the same mechanics as a basic starter game for myself.

The whole idea for the game was a starting school project to get the idea and the hangs of game designing. In my eyes it went quite well, but it did bring some problems in my everyday life to adjust the timing of certain projects and the game. In the end, I found this being a success for a starter project. Later along the line I try to improve this game and upcomings aswell.
